# dotfiles

#### Install softwares
* Debian/Ubuntu
```
sudo apt-get install ack aria2 cmake curl g++ gcc git vim wget zsh
```

* Mac OS X
```
brew install vim
brew install zsh
```

#### Configure vim
```
wget "https://raw.githubusercontent.com/fabioperez/dotfiles/master/dottools/vimstall.sh"
bash vimstall.sh
rm -f vimstall.sh
```
